# Лови/Избегай — мини-аркада (Web page + Canvas)

## Механика
Персонаж стоит на полу и двигается стрелками ← →.
Сверху падают предметы:
- хорошие: +1 (и комбо множитель)
- плохие: -1 (и опционально -жизнь)

## Запуск в VS Code
1) Распакуй zip
2) VS Code → File → Open Folder… → выбери папку проекта
3) Поставь расширение **Live Server**
4) Открой `index.html` → нажми **Go Live**

## Управление
- ← / → движение
- Space пауза
- D дебаг (показывает хитбоксы)

## Как добавить свои спрайты
Положи файлы в `assets/` и назови так (или поменяй пути в `config.js`):
- `background.jpg` (16:9)
- `player.png`
- `good1.png ... good5.png` (5 позитивных предметов)
- `bad1.png ... bad3.png` (негативные предметы)

Если спрайтов нет — игра рисует формы (всё равно работает).

## Настройки баланса
В `config.js`:
- `rules.badTakesLife = true` чтобы плохие отнимали жизнь
- `physics.spawnBase` / `spawnMin` — частота спавна
- `physics.gravityRamp` — ускорение со временем
- `player.speed` — скорость игрока


## Эмоции персонажа
Добавь 3 спрайта в `assets/`:
- `player.png` — обычное
- `player_happy.png` — поймал хороший предмет
- `player_angry.png` — поймал плохой

Длительность эмоции: `playerMoodMs` в `config.js` (по умолчанию 260мс).
